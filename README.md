# Wavex

[![Hex.pm](https://img.shields.io/hexpm/v/wavex.svg?style=flat-square)](https://hex.pm/packages/wavex)

Read .WAVE files.

Currently only the PCM format is supported.

## Installation

1.  Add `wavex` to your `mix.exs` dependencies:

```elixir
def deps do
  [
    {:wavex, "~> 0.1.0"}
  ]
end
```

2.  Update your dependencies:

```elixir
$ mix deps.get
```
